## 三、关键发现

### 1. CoT 对选择题适得其反

这是本周最重要的发现。实验数据清晰表明：
- v1.0（简洁输出）= 62.5%
- v1.1（CoT 推理）= 43.5%（下降 19%）

**教训**：CoT 应该选择性使用，不是所有题型都需要推理过程。对于选择题，直接输出选项字母是最佳策略。

### 2. 模型擅长和短板

**擅长**：
- 历史/政治（记忆类）：> 80%
- 化学概念判断：74%
- 英语短语辨析：59%

**短板**：
- 语文错别字/文言/文体：38%
- 数学计算/概念：47%
- 物理估算：46%（math 得分低部分因题库噪声）

### 3. 评测体系价值

200 题 Benchmark + 双指标体系提供了客观的模型评估手段：
- 不再依赖 10 题手动测试的偶然性
- Error Recovery Score 可量化迭代改进
- Error Set 为第三周错误分析工作奠定了基础
